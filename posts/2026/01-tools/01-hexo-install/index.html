Hello Hexo with NexT

发表于2024-01-01|更新于2026-01-28|test
|浏览量:

欢迎使用基于 NexT 主题的 Hexo 博客！

功能特性

  • ✅ 响应式设计
  • ✅ 代码高亮
  • ✅ 优雅的阅读体验
  • ✅ 丰富的配置选项

开始写作

使用 hexo new "文章标题" 创建新文章。

主题配置

NexT 主题已配置为 Pisces 风格，包含以下菜单：

  • 🏠 首页
  • ❤️ Chocolate（生活感悟）
