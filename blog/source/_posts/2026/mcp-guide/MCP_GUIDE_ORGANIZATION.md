# MCP 指南内容整理说明

## 问题描述
发现两个目录包含重复的 MCP 指南内容：

1. `/blog/source/_posts/2026/01-tools/mcp-guide/` - 技术实现导向的 MCP 指南
2. `/blog/source/_posts/2026/mcp-guide/` - 概念应用导向的 MCP 指南

## 内容对比
- 两个目录都包含 MCP 基础、AI Agents、Skills 系统等内容
- 但侧重点不同：
  - 01-tools/mcp-guide: 更偏向技术实现、代码示例、架构细节
  - mcp-guide: 更偏向概念解释、应用场景、生活类比

## 建议处理方式
保留两套内容，因为它们服务于不同的读者群体：
- 技术人员更适合阅读 01-tools/mcp-guide 中的技术细节
- 业务人员和初学者更适合阅读 mcp-guide 中的概念解释

## 维护注意事项
1. 当更新 MCP 相关内容时，需要同时考虑两个版本
2. 保持两个版本内容的一致性（核心概念方面）
3. 在 README 和导航中明确区分两个版本的目标受众
