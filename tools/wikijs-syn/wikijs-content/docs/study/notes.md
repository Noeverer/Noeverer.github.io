# 学习笔记

学习过程中的笔记和总结。

## 学习方法

### 费曼学习法

1. 选择一个概念
2. 用简单的语言解释给他人听
3. 找出理解不足的地方
4. 回顾和简化

### 番茄工作法

- 25 分钟专注学习
- 5 分钟休息
- 4 个番茄钟后休息 15-30 分钟
